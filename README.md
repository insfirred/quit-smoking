# Project Setup
### This project is built in Flutter v3.3.2
##### Fork the repository and Clone it
```bash
git clone https://github.com/insfirred/inshorts-clone.git
```
  ##### Move to the desired folder
```bash
cd inshorts_clone
```

  ##### To run the app, simply write
```bash
flutter pub get
```

```bash
flutter run
```
